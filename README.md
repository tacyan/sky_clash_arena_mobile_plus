# Sky Clash Arena Mobile Plus

Image-genで生成したキャラクター/UI素材を使った、スマホ操作強化版のスマブラ風2.5Dアクションゲームです。

## 起動

```bash
python3 -m http.server 5173
```

```text
http://localhost:5173
```

## 操作

### PC

- 移動: A/D または ←/→
- ダッシュ: Shift
- ジャンプ: W / ↑ / Space
- 通常攻撃: J / X
- ガード: L / ↓
- 必殺: K / C
- ポーズ: P
- タイトルへ戻る: Esc
- リスタート: R

### スマホ

- 左下: ◀ / ▶ / DASH
- 右下: JUMP / ATTACK / GUARD / SKILL

横画面推奨です。

## 追加内容

- 剣士・魔導士スプライトを最新Image-gen素材へ差し替え
- 生成画像のチェッカー背景を透過処理
- HUD/キャラ選択用ポートレートをImage-gen素材へ差し替え
- スマホボタンを大型化・押しやすく調整
- ダッシュ追加
- ガード追加
- ガードゲージ追加
- ガードブレイク追加
- キャラ別必殺技演出追加
  - 剣士: 蒼刃ダッシュ斬り
  - 獣: 地割れスマッシュ
  - 格闘家: 旋風蹴り
  - 魔導士: 紫電魔法弾

## 補足

ブラウザで `index.html` を直接開くと画像読み込みが安定しない場合があります。必ずローカルサーバー経由で起動してください。
